# 学习地图 PRD v2（2026-07-23）

## 产品哲学（所有功能取舍的判据）
1. **不诊断，不推荐** —— 没有诊断依据的内容推送都是噪音
2. **输出必须可执行** —— 每条建议都要能变成今天能打卡的动作
3. **为迁移而教** —— 课内会 ≠ 考试会，每个知识点必须有变式
4. **一次只做一件事** —— 家长每次打开，只面对一个决策

## 用户与定位
- 用户：K12 家长（决策与监督者），学生是执行者；教培老师不是目标用户
- 定位：家长的学习导航工具。不卖课、不做资源库、不卖信息差；做国家平台的"导购层"
- 商业模式：订阅会员制，300-600 元/年，按月可退，无"终身"话术，不接课程返佣
- 合规红线：无真人答疑/督学/批改；文案无"辅导/授课/提分"；解析给思路不给答案；资源跳转/内嵌官方平台并标注来源

## 本次迭代需求（来自真实用户反馈 + 对标调研）

### R1 UI 重做：tab 化 + Marble 式交互
- 四个底部 tab（移动端优先）：**诊断 / 计划 / 图谱 / 我的**
- 主入口是"诊断"，不是地图（Khan 教训：用户要线性路径，地图是诊断视图）
- 图谱 tab 重做视觉：深色底（#0A0C14）、默认边透明度 ≤0.1、慢速自转（悬停/触摸即停）、入场生长动画、点选节点 → 相机聚焦 + 先修链高亮 + 无关节点降至 4% 透明度 + 侧栏（移动端为底部抽屉）显示：先修总数（传递闭包）、直接先修、解锁什么
- 节点详情含"你孩子能做到……吗"问句（来自 coaching 数据）
- 不引入 three.js/d3，保持 vanilla JS + canvas 纯静态

### R2 诊断输出 = 可打卡计划
- 诊断结果不再是一段建议文字，而是计划清单：每项 = 动作 + 预计分钟 + 验收标准 + 关联资源/练习，可勾选打卡，存 localStorage
- "计划"tab 展示今日清单 + 完成进度 + 复测入口

### R3 站内看课（不跳转）
- 课程页内嵌 iframe 加载官方课时（平台无 X-Frame-Options 限制，已验证），外层包我们自己的 UI：配套任务单/练习入口、看完打卡按钮
- 页面标注"课程资源来自国家中小学智慧教育平台"

### R4 内容原子升级（试点：初中语文文言文）
新增 data/coaching.json，按 topicId  keyed：
```json
{
  "topicId": {
    "ask": "你孩子能做到……吗（可观察行为问句）",
    "pattern": "规律总结，如：名词/代词 + 之 + 动词 = 宾语前置标志",
    "variants": [{"text": "课外变式例句，如：句读之不解，惑之不解", "explain": "为什么这仍然是同一个规律"}],
    "drills": [{"q": "同类训练题", "answer": "答案", "hint": "给思路不给答案的提示"}]
  }
}
```
- 试点范围：初中语文文言文语法相关节点（宾语前置、状语后置、判断句、被动句、词类活用、古今异义、一词多义、虚词用法等，约 15-25 个节点）
- dependencies.json 的边补 `reason` 字段（试点子图即可）
- 诊断建议文案原则：给家长的话必须"看得懂、今天能执行"

### R5 资源管线扩展（课程包五件套）
- 从现有课时视频深链扩展到：视频课程 / 课件 / 教学设计 / 学习任务单 / 课后练习
- 产出 data/resources_pack.json，按知识点关联
- 优先利用 data/raw_smartedu/ 已有缓存；需要登录态的接口标注出来，不硬爬

## Loop 验收目标 v2.1（2026-07-23，来自真机走查）

主验收视口：移动端 360×782（iPhone）。桌面端 1280×800 回归不破。

- **L1 诊断答题**：答完一题自动进入下一题；可点"上一题"返回修改答案；全部答完后结果**自动标记到图谱节点掌握度**，无需手动操作
- **L2 诊断结果页**："生成打卡清单"为底部悬浮按钮，任何滚动位置可见可点
- **L3 图谱移动端**：默认只显示当前选中学科并 fit-to-view 一屏看清；支持双指缩放（以双指中点为锚点）和单指拖拽；节点点击热区≥28px；canvas 占可用高度≥70%；列头不依赖世界坐标顶部；桌面端行为不变
- **L4 看课页**：彻底放弃 iframe 内嵌（国家平台有主动反嵌套保护：frame-busting 脚本+官方 App 走 SmartCanWebView UA 白名单，普通网页无法绕开，已实测验证）。改为**原生课程页**：用 resources_pack.json 元数据渲染课程卡片（封面、标题、时长、五件套清单），点播放跳官方页，返回后可"看完打卡"。任何资源不得再出现白屏
- **L5 计划 tab**：今日清单+进度+复测入口可用，打卡状态刷新不丢
- **L6 自动化验收**：puppeteer 脚本跑通 L1-L5 全链路（移动端视口），console 零报错，关键页面截图目检存档

# 学习地图 · 产品哲学/价值/目标 v3（2026-07-23）

输入：教育者观点调研（刘长铭/马景林/胡小群/李希贵/魏书生）、商业化审计、内容重组管线可行性、真实用户反馈、真机走查。

## 一、产品哲学 v3（六条）

前四条不变：**不诊断不推荐 / 输出必须可执行 / 为迁移而教 / 一次只做一件事**。新增：

5. **复杂度归产品，零备课归家长。**
   来源：李希贵"多帮少管，供给资源"+ 胡小群"被家长辅导毁掉的孩子不计其数"。家长不需要懂教学，备课、找材料、想方法这些复杂的事是产品的活，家长拿到的永远是"打开就能用"的东西。

6. **交付习惯回路，不只交付内容。**
   来源：魏书生"教育最高境界是自育自学"。内容交付完不等于学习发生。每个学习单元必须自带回路：做 → 打卡 → 反馈 → 下一步，让孩子逐步离开家长。

## 二、产品价值（一句话）

**家长每天 10 分钟，按清单执行就行——孩子卡在哪、今天干什么、用什么材料，我们想清楚了。**

- 对标的焦虑来源：马景林"最可怕的不是学生不努力，而是重复、没效率的行为占满课外时间"——我们消灭无效重复，每一分钟都花在真正的断点上。
- 内容标准：胡小群"学一道题，会一类题"——每个知识点必须有规律总结+课外变式，不接受课内记忆型学会。

## 三、第一性原理：学习到底怎么发生

拆到最底：**掌握 = 定位断点 → 针对性输入 → 变式训练 → 反馈巩固**，四步循环。

由此推出产品的核心价值单元不是图谱、不是资源链接，而是——

### 「补给包」：针对一个断点的、家长可执行的最小学习单元

- **5 分钟导学卡**：这个知识点是什么、孩子为什么错、规律一句话（家长看得懂的图文，不是 40 分钟视频）
- **3-5 道变式训练**：课内会≠考试会，练迁移
- **打卡 + 复测**：完成后自动进复习队列，隔几天回测

图谱是库存组织方式，诊断是匹配引擎，**补给包是交付物**，打卡是习惯回路。会员费买的是持续不断的补给包供给。

## 四、内容策略：原材料怎么用（合规红线已查清）

- 小鹅通 3.7 万份付费资料、国家平台视频，**重组后直接给会员看 = 侵犯改编权**（有刑事判例，转售课程已有人获刑）。"改写到看不出来"不是抗辩。
- 正确定位：原材料是**内部提效工具**——ASR 转录、LLM 分析题型和思路、辅助教研生产**原创**补给包内容。管线成本已验证：单课时处理 <0.5 元、3.7 万 PDF 打标挂接 430 图谱约 300-500 元。
- 所以内容资产路线 = 参考原材料 → 原创补给包。慢，但安全，且形成自己的资产，不被任何人卡脖子。

## 五、商业化差距（审计 P0 摘要）

1. **收不到钱**：激活码 VIP2026 硬编码在前端，查看源码即可白嫖；会员状态只存 localStorage，清缓存就没。→ 需要账号体系+真实支付，这是收费的前提性阻塞。
2. **首屏 3 秒测试失败**：新用户第一眼是一屏自转圆点，没有一句话说清这是什么。
3. **付费墙时机倒挂**：诊断这个 aha moment 被墙挡住。改为：首次诊断+报告免费，墙后移到"生成清单"和"补给包"。
4. **信任状为零**：无退款承诺、无团队介绍、无案例。
5. **内容深度**：只有道法+语文两科撑不起 300-600/年，早鸟价 99-199 过渡，或先扩数学。
6. **品牌名**："学习地图·K12家长导航"是品类描述不是品牌，且"K12"在广告投放里有合规风险，需要改名。

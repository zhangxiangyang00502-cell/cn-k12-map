## 一、这份提示词改变了什么：从「工具」到「教练」

我们之前建的是发动机：430 节点图谱、掌握度标记、诊断题库、打卡计划、内容库句柄。
提示词给的是驾驶舱：**会员买的不是工具，是"小满"这个教练**——孩子不问，它也驱动学习。

提示词里最关键的判断（正确）：**离开状态注入，它就退化成豆包。** 模型能力是公共的，护城河是注入的四样数据结构。盘点我们手里有什么：

| 状态注入 | 现状 | 差距 |
|---|---|---|
| 图谱库 | ✅ 430 节点 + 440 先修边 + reason | 成熟 |
| 内容库句柄 | ⚠️ 有 coaching（9 节点）+ 五件套元数据（420 条） | 补给包内容未生产，只有试点 |
| 掌握度模型 | ⚠️ 只有"还不会/已掌握"两态 | 需要 0-1 连续值 + 遗忘曲线（SM-2 变体）+ error_type 归因 |
| 任务卡引擎 | ⚠️ 计划 tab 是静态清单 | 缺"忆"（间隔回忆挑战）和"产出"（讲一遍/出题考别人）两步，无排程器 |

闭环五步对照：测 ✅ 已有 / 学 ⚠️ 只有资源链接没有最小输入卡 / 练 ⚠️ 变式题只有 9 个节点 / 忆 ❌ 缺失 / 产出 ❌ 缺失。

## 二、教练侧引擎说明书（用户提供的提示词，直接采用）

角色：小满，K12 伴学教练，非答疑机器人。目标不是"看过了"而是"能回忆、能迁移"。
五步闭环：测（先测后教）→ 学（最小必要输入）→ 练（变式不重复）→ 忆（间隔提取，严禁先给答案）→ 产出（掌握度达标后讲一遍/出题考别人）。
教学原则：提取优先、反口诀、一次一个漏洞、不押题不承诺分数。
家长报告三件套：能回忆什么、不能回忆什么、系统接下来怎么安排。禁止报告学习时长、禁止焦虑营销。

## 三、管线侧引擎说明书（补全另一侧）

```markdown
# 小满伴学 · 内容加工管线提示词 v0.1

## 角色
你是小满教研管线。输入是原材料（国家平台课程、资料库 PDF、课标文本），
输出是挂接到图谱节点的"补给包"。你不生产"资料汇编"，只生产可执行的学习单元。

## 输入
- 图谱节点：topicId、name、ask、pattern、先修关系、所属年级学科
- 原材料：按节点检索到的课程文字稿 / PDF 文本 / 课标要求

## 每个补给包必须包含（缺一件不出厂）
1. 导学卡：≤300 字，家长读得懂。结构=孩子为什么错 → 规律一句话 → 一个例子
2. 变式题 3-5 道：不重复课内原例，练迁移；每题带"给思路不给答案"的 hint
3. 回忆卡：一张，用于间隔挑战（问题在正面，答案在背面）
4. 产出任务：一个（如"用自己的话给家长讲一遍这个规律"）

## 质量红线
- 不许出现"辅导/授课/提分/押题/冲刺"字样
- 变式题必须考"课外材料上的迁移"，不许考课内记忆
- 所有事实必须能在原材料或课标中找到依据，禁止编造

## 出厂校验（不过则打回）
- 导学卡读一遍，五年级家长能听懂吗？
- 变式题答案唯一且可判分吗？
- 和该节点的先修关系矛盾吗？
```

## 四、政策约束下的 PMF 与商业化路径

政策边界（已查实）：不能有真人培训行为、不能给学生拍照出答案、国家平台资源只能链接。市场上学而思/猿辅导卖课卖硬件，国家平台免费发资源——**PMF 假设：家长愿意付一杯咖啡钱/月，买"自己不懂教学也能带孩子精准查漏补缺"的教练服务。**

商业化路径（按依赖排序）：
1. **MVP 验证（现在）**：免费诊断引流，验证"诊断→清单"的打开率和留存
2. **收第一笔钱（Loop 2）**：账号+微信支付，早鸟 ¥199 收 50 个付费家长——这是 PMF 真验证，在此之前所有"定位清楚"都只是假设
3. **内容规模化（Loop 3）**：管线批量生产补给包，从 10 个试点到覆盖语文+道法高频断点
4. **旗舰场景（Loop 4）**：拍整卷诊断，获客钩子

教练对话界面需要 LLM API → 需要后端 → 排在 Loop 2 账号体系之后。前端先把"任务卡"做成教练的化身（不需要对话也能驱动闭环）。

## 五、H5 体验三件事（用户真机反馈）

1. **护眼**：深色+小字不适合孩子。分工：家长侧（诊断结果/图谱/计划）保留深色高级感；**孩子侧（测一测答题/补给包学习）切浅色护眼模式**——暖白底 #FAF7F2、正文 ≥17px、行距 1.8、按钮加大。
2. **图谱炫酷**：vanilla canvas 到天花板了，承认。上 three.js（WebGL）做真 3D 星图：辉光、景深、星云粒子、聚焦飞行。图谱是给家长的"演示时刻"，是卖会员的 showcase，值得引入唯一一个依赖。
3. **交互架构**：从"四个 tab 的工具箱"改为"**今日任务卡**"驱动——打开就是教练排好的今日三件事（测/忆/练），图谱、诊断降为二级入口。这才像教练，不像工具。
